# Task Manager Frontend

## 📌 Overview

Frontend application for a task management system built with React.

## 🛠 Tech Stack

* React
* JavaScript (ES6+)
* Axios

## 📂 Structure

* pages: Page components
* components: Reusable components
* api: API communication layer

## 🚀 Features

* User list 조회
* REST API 연동
* CRUD UI (planned)

## 🔗 API

* Backend: http://localhost:8080

## ⚙️ Run

```
npm install
npm start
```

## 📌 Notes

* This project consumes REST APIs from Spring Boot backend.

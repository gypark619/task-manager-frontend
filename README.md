# Task Manager Frontend

## 📌 Overview

React 기반의 업무 관리 시스템 프론트엔드 애플리케이션입니다.

백엔드 REST API와 연동하여 사용자, 팀, 직급, 권한 관리 기능을 구현하였으며,  
검색, 페이징, 정렬, 상세 입력 폼 등 실무형 관리 화면 구조를 설계하였습니다.

또한 입력 UX 개선과 화면 상태 유지 구조를 통해 실제 업무 환경에 가까운 사용 경험을 제공하는 것을 목표로 하였습니다.

---

## 🛠 Tech Stack

- React
- JavaScript (ES6+)
- Axios

---

## 📂 Structure

- pages: 화면 단위 컴포넌트
- components: 재사용 UI 컴포넌트
- api: API 통신 모듈
- styles: 공통 스타일
- constants: 공통 상수

## 🚀 주요 기능

### 📌 관리 기능
- 사용자 / 팀 / 직급 / 권한 CRUD
- 사용자 권한 및 조직 정보 관리

### 📌 조회 기능
- 이름, 부서, 직급, 상태 조건 기반 검색
- 서버 API 기반 필터링 처리
- 페이징 및 정렬 기능 지원

## 🧩 UX 개선

### 목록-상세 연동
- 조회 시 첫 번째 row 자동 선택
- 선택 데이터 기반 상세 입력 폼 바인딩
- 미선택 시 상세 영역 비활성화

### 입력 흐름 개선
- 신규 row 생성 및 즉시 입력 가능
